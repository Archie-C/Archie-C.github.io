Python for Machine Learning.

The basic python skills needed for this course.

This topic covers the essential Python skills needed for the start of the course. It is not meant to be a complete Python tutorial, if there is anything that needs to be explained in more detail, I emplore you to research it yourself using any websearch or AI tool.

Setup

For the whole course, I will be using uv as the package manager. It's perfectly fine to use alternative package managers if you prefer, such as pip or conda, just be aware you will need to translate any examples into your management workflow. To install uv, you can go to uv docs.

Creating a virtual environment

With uv, creating a virtual environment is very simple. Just run uv venv in the terminal. You can optionally provide a name for the environment, otherwise it defaults to .venv. You can also specify the Python version to use with the --python flag, for example uv venv --python 3.11. You will also need to activate the virtual environment before you can use it.

Matrices and Vectors

I am assuming you are already familiar with basic data structures in Python, such as lists, tuples, dictionaries, and sets. As a reminder, we will use vectors, matrices, and tensors constantly in this course: 

my_vector = [1, 2, 3]                          # 1D
my_matrix = [[1, 2, 3], [4, 5, 6]]            # 2D
my_tensor = [[[1, 2], [3, 4]], [[5, 6], [7, 8]]]  # 3D

We can index lists using [] and slice them using :. 

my_vector = [1, 2, 3]

my_vector[0]    # 1   (first element)
my_vector[-1]   # 3   (last element)
my_vector[0:2]  # [1, 2]
my_vector[:2]   # [1, 2]
my_vector[1:]   # [2, 3]

We can also use list comprehensions to create new lists based on existing ones. 

my_vector = [1, 2, 3]
squared = [x**2 for x in my_vector]  # [1, 4, 9]

NumPy

While we can use lists to represent vectors and matrices, it is often more convenient (and more performant) to use the NumPy library.

First we must install NumPy:

uv pip install numpy

We can define a vector, matrix or tensor using a NumPy array in much the same way as before:

import numpy as np

my_vector = np.array([1, 2, 3])  # 1D array
my_matrix = np.array([[1, 2, 3], [4, 5, 6]])  # 2D array
my_tensor = np.array([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])  # 3D array

The line import numpy as np imports the NumPy library and gives it the alias np.
