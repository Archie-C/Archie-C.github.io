Probability & Statistics.

Probability theory provides the mathematical language to formalize uncertainty. In deep learning, probability guides how we represent noise, design loss functions, and reason about model confidence.

What this note covers

  • Random variables and probability densities (PDFs, PMFs, and CDFs)
  • Conditional probability, rules of probability, and Bayes' Theorem
  • Maximum Likelihood Estimation (MLE) and Maximum A Posteriori (MAP)
  • Summary statistics: Expectation, Variance, Covariance, and the Covariance Matrix
  • The Multivariate Gaussian and the Reparameterization Trick
  • A Python implementation of Cholesky-based Gaussian sampling

For a detailed treatment, see Chapter 6 of Mathematics for Machine Learning.

Random Variables and Densities

A random variable $X$ is a variable whose values are outcomes of a random phenomenon. We distinguish between two main types:

  • Discrete random variables: Take on values from a countable set. Their distribution is described by a Probability Mass Function (PMF), denoted $p(x) = P(X = x)$, which must satisfy $\sum_i p(x_i) = 1$.

  • Continuous random variables: Take on values in a continuous range (like $\mathbb{R}$). Because the probability of a continuous variable taking any single exact value is zero, we describe their distribution using a Probability Density Function (PDF), $p(x)$. The probability that $X$ lies in an interval $[a, b]$ is the integral of the PDF:

    $$ P(a \le X \le b) = \int_a^b p(x) \, dx $$

    The PDF must satisfy $p(x) \ge 0$ and $\int_{-\infty}^{\infty} p(x) \, dx = 1$.

For both types, the Cumulative Distribution Function (CDF), $F(x)$, describes the probability that the random variable is less than or equal to $x$:

$$ F(x) = P(X \le x) $$

connection to deep learning

In classification tasks, neural networks output raw real numbers called logits. We convert these logits $\mathbf{z}$ into a discrete probability distribution over classes using the softmax function:

$$ p(y = i \mid \mathbf{x}) = \frac{e^{z_i}}{\sum_{j=1}^C e^{z_j}} $$

This maps the output elements to the interval $(0, 1)$ such that they sum to 1, fulfilling the definition of a PMF.

Rules of Probability & Bayes' Theorem

Given two random variables $X$ and $Y$, their relationship is governed by two fundamental rules:

  • Sum Rule (Marginalization): Determines the distribution of a single variable by summing or integrating over the other:

    $$ p(x) = \int p(x, y) \, dy $$

  • Product Rule: Relates the joint probability $p(x, y)$ to the conditional probability $p(y \mid x)$:

    $$ p(x, y) = p(y \mid x) p(x) $$

Combining the product rule and the symmetry of joint probability ($p(x, y) = p(y, x)$) yields Bayes' Theorem:

$$ p(y \mid x) = \frac{p(x \mid y)p(y)}{p(x)} $$

In machine learning, we use Bayes' Theorem to estimate parameters $\boldsymbol{\theta}$ given observed data $\mathcal{D}$:

$$ p(\boldsymbol{\theta} \mid \mathcal{D}) = \frac{p(\mathcal{D} \mid \boldsymbol{\theta}) p(\boldsymbol{\theta})}{p(\mathcal{D})} $$

We interpret each component as follows:

  • Posterior $p(\boldsymbol{\theta} \mid \mathcal{D})$: Our updated belief about the parameters after observing the data.

  • Likelihood $p(\mathcal{D} \mid \boldsymbol{\theta})$: How likely the observed data is under the parameters.

  • Prior $p(\boldsymbol{\theta})$: Our initial belief about the parameters before seeing data.

  • Evidence $p(\mathcal{D})$: A normalization constant representing the probability of the data, computed as $\int p(\mathcal{D} \mid \boldsymbol{\theta}) p(\boldsymbol{\theta}) \, d\boldsymbol{\theta}$.

mle vs map

Because the evidence $p(\mathcal{D})$ is independent of $\boldsymbol{\theta}$, we can optimize parameters without computing it:

Maximum Likelihood Estimation (MLE): Maximizes only the likelihood:

$$ \boldsymbol{\theta}_{\text{MLE}} = \arg\max_{\boldsymbol{\theta}} \log p(\mathcal{D} \mid \boldsymbol{\theta}) $$

Maximum A Posteriori (MAP): Includes the prior, acting as a regularizer:

$$ \boldsymbol{\theta}_{\text{MAP}} = \arg\max_{\boldsymbol{\theta}} \left[ \log p(\mathcal{D} \mid \boldsymbol{\theta}) + \log p(\boldsymbol{\theta}) \right] $$

For instance, assuming a Gaussian prior on weight parameters yields $L_2$ regularization (weight decay).

Summary Statistics

Rather than working with entire distributions, we often summarize them using key statistics:

The Expected Value (or mean) is the average value of a random variable, weighted by its probability:

$$ \mathbb{E}[X] = \int x p(x) \, dx $$

Expectation is a linear operator: $\mathbb{E}[aX + bY] = a\mathbb{E}[X] + b\mathbb{E}[Y]$.

The Variance measures the spread of the distribution around its mean:

$$ \text{Var}[X] = \mathbb{E}\left[ (X - \mathbb{E}[X])^2 \right] = \mathbb{E}[X^2] - (\mathbb{E}[X])^2 $$

For multiple dimensions, Covariance measures how two random variables vary together:

$$ \text{Cov}[X, Y] = \mathbb{E}\left[ (X - \mathbb{E}[X])(Y - \mathbb{E}[Y]) \right] $$

The covariance of a random vector $\mathbf{x} \in \mathbb{R}^d$ is represented by the symmetric Covariance Matrix $\mathbf{\Sigma} \in \mathbb{R}^{d \times d}$:

$$ \mathbf{\Sigma} = \mathbb{E}\left[ (\mathbf{x} - \boldsymbol{\mu})(\mathbf{x} - \boldsymbol{\mu})^\top \right] $$

where the diagonal entries $\Sigma_{ii}$ represent the variance of each component, and the off-diagonal entries $\Sigma_{ij}$ represent the covariances.

The Multivariate Gaussian

The univariate Gaussian distribution $\mathcal{N}(\mu, \sigma^2)$ is parameterized by its mean $\mu$ and variance $\sigma^2$. Scaling to $d$ dimensions, the Multivariate Gaussian distribution $\mathcal{N}(\boldsymbol{\mu}, \mathbf{\Sigma})$ has the PDF:

$$ p(\mathbf{x}) = \frac{1}{(2\pi)^{d/2} |\mathbf{\Sigma}|^{1/2}} \exp\left( -\frac{1}{2} (\mathbf{x} - \boldsymbol{\mu})^\top \mathbf{\Sigma}^{-1} (\mathbf{x} - \boldsymbol{\mu}) \right) $$

where $\boldsymbol{\mu} \in \mathbb{R}^d$ is the mean vector and $\mathbf{\Sigma} \in \mathbb{R}^{d \times d}$ is the symmetric, positive-definite covariance matrix.

The Reparameterization Trick

In deep learning, we often want to train models that output random variables (such as in Variational Autoencoders). However, sampling a value $\mathbf{x} \sim \mathcal{N}(\boldsymbol{\mu}, \mathbf{\Sigma})$ directly is a stochastic node, which means we cannot backpropagate gradients through it because the derivative of a random sample with respect to the parameters ($\boldsymbol{\mu}$ and $\mathbf{\Sigma}$) is undefined.

To bypass this, we use the reparameterization trick. We isolate the stochasticity by sampling standard normal noise $\mathbf{z} \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$ and then construct $\mathbf{x}$ deterministically:

$$ \mathbf{x} = \boldsymbol{\mu} + \mathbf{L}\mathbf{z} $$

where $\mathbf{L}$ is the lower-triangular Cholesky factor of the covariance matrix, satisfying $\mathbf{\Sigma} = \mathbf{L}\mathbf{L}^\top$. Because the random node $\mathbf{z}$ has no parameters, the mapping from $\boldsymbol{\mu}$ and $\mathbf{L}$ to $\mathbf{x}$ is completely differentiable, allowing standard backpropagation to flow through!

Implementation: Multivariate Sampling

Below is a complete implementation of Multivariate Gaussian sampling using the Box-Muller transform for standard normal noise and Cholesky decomposition. We compute the empirical statistics of 10,000 samples to verify convergence back to the target parameters.

First, let's set up the environment:

uv venv --python 3.13
# We will use numpy to calculate empirical mean and covariance for verification
uv pip install numpy

We write our sampling and verification script:

import math
import numpy as np

def box_muller() -> tuple[float, float]:
    """Generates two independent standard normal samples U1, U2 ~ N(0, 1)."""
    # Math.random() equivalent in python
    import random
    u1 = random.random()
    u2 = random.random()
    # Avoid log(0)
    while u1 == 0.0:
        u1 = random.random()
        
    r = math.sqrt(-2.0 * math.log(u1))
    theta = 2.0 * math.pi * u2
    return r * math.cos(theta), r * math.sin(theta)

def cholesky_2d(sigma_x: float, sigma_y: float, rho: float) -> list[list[float]]:
    """
    Computes the 2x2 Cholesky factor L for the covariance matrix:
    [[sigma_x^2,           rho * sigma_x * sigma_y],
     [rho * sigma_x * sigma_y,           sigma_y^2]]
    L = [[L11, 0],
         [L21, L22]]
    """
    l11 = sigma_x
    l21 = rho * sigma_y
    l22 = sigma_y * math.sqrt(1 - rho * rho)
    return [[l11, 0.0], [l21, l22]]

def sample_multivariate_gaussian(
    mu_x: float, mu_y: float, 
    sigma_x: float, sigma_y: float, 
    rho: float, num_samples: int
) -> np.ndarray:
    """Generates samples from N(mu, Sigma) using Box-Muller and Cholesky."""
    L = cholesky_2d(sigma_x, sigma_y, rho)
    samples = []
    
    for _ in range(num_samples // 2):
        # Get two standard normal noise variables
        z1, z2 = box_muller()
        
        # Apply scaling and shift
        x1 = mu_x + L[0][0] * z1
        y1 = mu_y + L[1][0] * z1 + L[1][1] * z2
        
        x2 = mu_x + L[0][0] * z2
        y2 = mu_y + L[1][0] * z2 + L[1][1] * z1
        
        samples.append([x1, y1])
        samples.append([x2, y2])
        
    return np.array(samples)

if __name__ == "__main__":
    # Target distribution parameters
    mu_x, mu_y = 0.5, -1.0
    sigma_x, sigma_y = 1.5, 0.8
    rho = 0.6
    
    print("Generating 10,000 samples...")
    samples = sample_multivariate_gaussian(mu_x, mu_y, sigma_x, sigma_y, rho, 10000)
    
    # Verify empirical statistics
    empirical_mean = np.mean(samples, axis=0)
    empirical_cov = np.cov(samples, rowvar=False)
    
    target_cov = np.array([
        [sigma_x**2, rho * sigma_x * sigma_y],
        [rho * sigma_x * sigma_y, sigma_y**2]
    ])
    
    print(f"\nTarget Mean:    {[mu_x, mu_y]}")
    print(f"Empirical Mean: {np.round(empirical_mean, 4).tolist()}")
    print("\nTarget Covariance:")
    print(target_cov)
    print("Empirical Covariance:")
    print(empirical_cov)

Executing this script demonstrates that shifting and scaling standard normal variables by the Cholesky factor of the covariance matrix generates sample statistics that converge exactly to their target values!
