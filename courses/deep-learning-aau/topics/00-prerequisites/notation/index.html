Notation.

The mathematical notation used throughout the notes.

1. Sets & General Math

Sets

SymbolMeaning
$\mathbb{R}$Real numbers
$\mathbb{R}^n$Real n-dimensional space
$\mathbb{Z}$Integers
$\mathbb{N}$Natural numbers
$\mathcal{X}$Input space (calligraphic for general sets)
$\mathcal{Y}$Output space
$x \in \mathcal{X}$x is a member of set 𝒳
$|\mathcal{S}|$Cardinality of set 𝒮

Intervals & Ranges

SymbolMeaning
$[a, b]$Closed interval
$(a, b)$Open interval
$[n]$Shorthand for {1, …, n}

General Math

SymbolMeaning
$:=$Equality by definition
$\propto$Proportional to
$\approx$Approximately equal
$\mathbb{1}[\mathrm{condition}]$Indicator function, e.g. 𝟙[y = 1]

Norms

SymbolMeaning
$\|x\|_1$L1 norm
$\|x\|_2$ or $\|x\|$L2 norm
$\|x\|_p$Lp norm
$\|A\|_F$Frobenius norm

Functions

SymbolMeaning
$f : \mathcal{X} \to \mathcal{Y}$Function f mapping from 𝒳 to 𝒴
$f \circ g$Composition of f and g
$\arg\max$Argument of the maximum
$\arg\min$Argument of the minimum

2. Data & Indexing

Sample & Dimensionality

SymbolMeaning
$n$Number of training samples
$d$Input dimensionality
$K$Number of classes
$L$Number of layers

Samples & Features

SymbolMeaning
$x \in \mathbb{R}^d$A single input vector
$y$A single output / label
$x_i$The i-th training sample
$x_{i,j}$The j-th feature of sample i
$\mathcal{D} = \{(x_i, y_i)\}_{i=1}^n$The full dataset

The Design Matrix

SymbolMeaning
$\mathbf{X} \in \mathbb{R}^{n \times d}$Design matrix — rows are samples, columns are features
$x_i^\top$The i-th row of X (one sample)
$x_j$The j-th column of X (one feature)

Indexing Conventions

SymbolMeaning
$i$Indexes samples (i = 1, …, n)
$j$Indexes features (j = 1, …, d)
$k$Indexes classes (k = 1, …, K)
$t$Indexes time steps
$l$Indexes layers

3. Linear Algebra

Typesetting Conventions

SymbolMeaning
$a, x, \lambda$Scalar — lowercase italic
$\mathbf{x}, \mathbf{w}, \mathbf{b}$Vector — lowercase bold
$\mathbf{X}, \mathbf{W}, \mathbf{A}$Matrix — uppercase bold
$\mathcal{T}$Tensor — uppercase calligraphic

Operations

SymbolMeaning
$\mathbf{A}^\top$Transpose
$\mathbf{A}^{-1}$Inverse
$\mathbf{A}^{+}$Pseudoinverse (Moore–Penrose)
$\det(\mathbf{A})$ or $|\mathbf{A}|$Determinant
$\operatorname{tr}(\mathbf{A})$Trace
$\mathbf{x}^\top\mathbf{y}$ or $\langle \mathbf{x}, \mathbf{y} \rangle$Dot product
$\mathbf{x} \odot \mathbf{y}$Element-wise (Hadamard) product
$\mathbf{A} \otimes \mathbf{B}$Kronecker product

Eigendecomposition & SVD

SymbolMeaning
$\lambda$Eigenvalue
$\mathbf{v}$Eigenvector
$\mathbf{A} = \mathbf{V}\Lambda\mathbf{V}^\top$Eigendecomposition
$\sigma_1 \ge \sigma_2 \ge \cdots \ge 0$Singular values in descending order
$\mathbf{A} = \mathbf{U}\Sigma\mathbf{V}^\top$Singular value decomposition

Shapes & Subspaces

SymbolMeaning
$\mathbb{R}^{n \times d}$Shape of a matrix — n rows, d columns
$\operatorname{rank}(\mathbf{A})$Rank of matrix A
$\operatorname{col}(\mathbf{A})$Column space of A
$\operatorname{null}(\mathbf{A})$Null space of A

4. Probability

Random Variables & Distributions

SymbolMeaning
$X, Y$Random variable — uppercase
$x, y$Realisation of a random variable — lowercase
$P(X = x)$Probability of an event
$p(x)$Probability density function
$P(x)$Probability mass function
$p(x, y)$Joint distribution
$p(y \mid x)$Conditional distribution
$p(x) = \int p(x, y)\,dy$Marginal distribution

Expectations & Moments

SymbolMeaning
$\mathbb{E}[X]$Expectation of X
$\mathbb{E}_{x \sim p}[f(x)]$Expectation of f(x) under distribution p
$\operatorname{Var}(X)$ or $\sigma^2$Variance
$\sigma$Standard deviation
$\operatorname{Cov}(X, Y)$Covariance between X and Y
$\boldsymbol{\Sigma}$Covariance matrix

Common Distributions

SymbolMeaning
$\mathcal{N}(\mu, \sigma^2)$Univariate Gaussian with mean μ and variance σ²
$\mathcal{N}(\boldsymbol{\mu}, \boldsymbol{\Sigma})$Multivariate Gaussian with mean μ and covariance Σ
$\operatorname{Bern}(p)$Bernoulli distribution with parameter p
$\operatorname{Cat}(\boldsymbol{\pi})$Categorical distribution with probabilities π
$\operatorname{Dir}(\boldsymbol{\alpha})$Dirichlet distribution with concentration α
$\operatorname{Beta}(\alpha, \beta)$Beta distribution with parameters α, β

Information Theory

SymbolMeaning
$H(X)$ or $H(p)$Entropy of X or distribution p
$H(p, q)$Cross-entropy between p and q
$D_{\mathrm{KL}}(p \| q)$KL divergence from q to p
$I(X; Y)$Mutual information between X and Y

5. ML Specific Notation

Parameters & Models

SymbolMeaning
$\boldsymbol{\theta}$Model parameters (general)
$\mathbf{w}$Weight vector
$b$Bias term
$\Theta$Parameter space
$\hat{\boldsymbol{\theta}}$Estimated / learned parameters
$f_{\boldsymbol{\theta}}(x)$Model function parameterised by θ
$\mathcal{H}$Hypothesis class

Loss & Risk

SymbolMeaning
$\ell(\hat{y}, y)$Loss function
$L(\boldsymbol{\theta})$Empirical risk — $\frac{1}{n}\sum_i \ell(f_{\boldsymbol{\theta}}(x_i), y_i)$
$R(\boldsymbol{\theta})$Expected risk — 𝔼[ℓ(f_θ(x), y)]
$\Omega(\boldsymbol{\theta})$Regularisation term
$\lambda$Regularisation strength

Predictions

SymbolMeaning
$\hat{y}$Predicted output
$y$True label
$\hat{p}$Predicted probability
$f_{\boldsymbol{\theta}}(x) = 0$Decision boundary

Optimisation

SymbolMeaning
$\nabla_{\boldsymbol{\theta}} L(\boldsymbol{\theta})$Gradient of loss with respect to θ
$\mathbf{H}$ or $\nabla^2 L$Hessian matrix
$\eta$Learning rate
$t$Training iteration / step
$\boldsymbol{\theta} \leftarrow \boldsymbol{\theta} - \eta\nabla L(\boldsymbol{\theta})$Gradient update rule

Regularisation

SymbolMeaning
$\lambda\|\boldsymbol{\theta}\|^2$L2 penalty
$\lambda\|\boldsymbol{\theta}\|_1$L1 penalty
$\lambda$Regularisation hyperparameter

6. Neural Network Notation

Architecture

SymbolMeaning
$L$Total number of layers
$d_l$Width (number of units) at layer l
$0$Index of input layer
$L$Index of output layer

Weights & Biases

SymbolMeaning
$\mathbf{W}^{(l)} \in \mathbb{R}^{d_l \times d_{l-1}}$Weight matrix at layer l
$\mathbf{b}^{(l)} \in \mathbb{R}^{d_l}$Bias vector at layer l
$\boldsymbol{\theta} = \{\mathbf{W}^{(l)}, \mathbf{b}^{(l)}\}_{l=1}^L$All parameters collected

Activations

SymbolMeaning
$\mathbf{z}^{(l)} = \mathbf{W}^{(l)}\mathbf{a}^{(l-1)} + \mathbf{b}^{(l)}$Pre-activation at layer l
$\mathbf{a}^{(l)} = \sigma(\mathbf{z}^{(l)})$Post-activation at layer l
$\mathbf{a}^{(0)} = \mathbf{x}$Input layer activations
$\mathbf{a}^{(L)} = \hat{y}$Output layer activations
$\sigma(\cdot)$Activation function (general)

Gradients in Backpropagation

SymbolMeaning
$\boldsymbol{\delta}^{(l)}$Error signal — loss gradient w.r.t. pre-activation at layer l
$\frac{\partial L}{\partial \mathbf{W}^{(l)}}$Loss gradient w.r.t. weight matrix at layer l
$\frac{\partial L}{\partial \mathbf{b}^{(l)}}$Loss gradient w.r.t. bias vector at layer l

Attention & Transformers

SymbolMeaning
$\mathbf{Q}$Query matrix
$\mathbf{K}$Key matrix
$\mathbf{V}$Value matrix
$\mathbf{A} = \operatorname{softmax}\left(\frac{\mathbf{Q}\mathbf{K}^\top}{\sqrt{d_k}}\right)$Attention scores
$d_k$Dimension of keys and queries
$H$Number of attention heads

7. Reinforcement Learning

Core RL Objects

SymbolMeaning
$s \in \mathcal{S}$State from state space 𝒮
$a \in \mathcal{A}$Action from action space 𝒜
$r \in \mathbb{R}$Reward
$t$Time step
$\tau = (s_0, a_0, r_0, s_1, \ldots)$Trajectory / episode

Policy

SymbolMeaning
$\pi(a \mid s)$Stochastic policy — probability of action a in state s
$a = \pi(s)$Deterministic policy
$\pi^\ast$Optimal policy

Value Functions

SymbolMeaning
$V^\pi(s)$State value function under policy π
$Q^\pi(s, a)$Action value function under policy π
$V^\ast(s)$Optimal state value function
$Q^\ast(s, a)$Optimal action value function
$A^\pi(s, a) = Q^\pi(s, a) - V^\pi(s)$Advantage function

Returns & Discounting

SymbolMeaning
$G_t = r_t + \gamma r_{t+1} + \gamma^2 r_{t+2} + \cdots$Discounted return
$\gamma \in [0, 1]$Discount factor
$\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)$Temporal difference (TD) error

Environment

SymbolMeaning
$p(s' \mid s, a)$Transition dynamics
$r(s, a)$Reward function
$T$Finite horizon
$\alpha$Learning rate for value function updates (RL convention)
