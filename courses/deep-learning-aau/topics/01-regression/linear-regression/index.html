Linear Regression.

Linear regression is perhaps the most fundamental machine learning algorithm. It underpins many of the concepts in this course and it's a great starting model to try as a benchmark on a new problem.

What this note covers

  • The model
  • The objective function
  • Implementation details

For a more formal definition, see Deep learning: Foundations and Concepts

The model

Below we have some data from a fictional school experiment where we measured the height and ages of students. A common task in school would be to ask the students to draw a line of best fit to model the relationship.

Now obviously we can see that there is a clear linear relationship between the age and the height of the students, and we can draw a line of best fit. But the question is, how do we find the best line?

Defining the line

You should be familiar with the equation of a line from school, which is given by:

$$ y = mx + c $$

where $m$ is the slope and $c$ is the y-intercept.

Here we want a more general formulation so that later we can model the relationship between multiple features and the target variable. So instead lets rewrite the equation as:

$$ y = w_1 x + w_0 $$

important

The notation changes, but the idea has not: $w_1$ is still the slope and $w_0$ is still the intercept. We use $w$ because the same notation will scale to many input features.

Fitting the line

Now that we have the equation of the line, we need to find the optimal values for the weights $w_0$ and $w_1$.

To do this, we define a loss function that measures how close our line is to the data points. We can then minimise this function to find the optimal parameters. For this example, the loss function is quite trivial, we just want to minimise the distance between the line and the data points.

In practice, we often use the squared error loss function:

$$ \mathcal{L} = \sum_{i=1}^n (y_i - \hat{y}_i)^2 $$

where $y_i$ is the true value and $\hat{y}_i$ is the predicted value from our model.

Nicely, the minimisation of this function has a closed form solution, which means we can find the optimal parameters without having to use gradient descent or any other iterative optimisation tricks.

Deriving the 1D solution

Since we have two parameters, we need to find the minimum of this function with respect to both $w_0$ and $w_1$. We can do this by taking the partial derivatives of the loss function with respect to each parameter and setting them to zero.

First, let's write the function in terms of the weights:

$$ \mathcal{L} = \sum_{i=1}^n (y_i - (w_1 x_i + w_0))^2 $$

Now we take partial derivatives:

$$ \begin{aligned} \frac{\partial \mathcal{L}}{\partial w_0} &= -2 \sum_{i=1}^n (y_i - (w_1 x_i + w_0)) = 0 \\ \frac{\partial \mathcal{L}}{\partial w_1} &= -2 \sum_{i=1}^n x_i (y_i - (w_1 x_i + w_0)) = 0 \end{aligned} $$

important

If you are not familiar with partial derivatives, now would be a good time to review them. We will use them frequently in the course.

These are two simulataneous equations that we can solve by first dropping the $-2$ and then distributing the summation:

$$ \begin{aligned} \sum_{i=1}^n y_i - \sum_{i=1}^n w_1 x_i - \sum_{i=1}^n w_0 = 0 \\ \sum_{i=1}^n x_i y_i - \sum_{i=1}^n w_1 x_i^2 - \sum_{i=1}^n w_0 x_i = 0 \end{aligned} $$

Lets work on the first equation:

If we rearrange the first equation for $w_0$, we get:

$$ w_0 = \frac{1}{n} \sum_{i=1}^n y_i - w_1 \frac{1}{n} \sum_{i=1}^n x_i $$

note that $\frac{1}{n} \sum_{i=1}^n x_i$ is just the mean of the $x_i$ values, often denoted as $\bar{x}$. Similarly, $\frac{1}{n} \sum_{i=1}^n y_i$ is the mean of the $y_i$ values, denoted as $\bar{y}$.

So we can rewrite the equation as:

$$ w_0 = \bar{y} - w_1 \bar{x} $$

Which is a much nicer form than what we had before. Now let's tackle the second equation. We can simplify the $\sum_{i=1}^n x_i$ as $n\bar{x}$ and then substitute our expression for $w_0$:

$$\sum_{i=1}^n x_i y_i - w_1 \sum_{i=1}^n x_i^2 - (\bar{y} - w_1 \bar{x}) n \bar{x} = 0$$

Then with some rearrangement, which I will leave out as an exercise. We are left with two analytical solutions for $w_0$ and $w_1$.

$$w_1 = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n (x_i - \bar{x})^2}$$

$$w_0 = \bar{y} - w_1 \bar{x}$$

Implementation

Now that we have the analytical solution, we can implement it in code. First, set up a virtual python environment:

uv venv --python 3.13

This just makes sure we have a clean environment to work in.

important

I use uv as my environment manager, but you can use any environment manager of your choice.

Then we can install the necessary dependencies:

uv pip install numpy matplotlib

and import them at the top of our .py file

from abc import ABC, abstractmethod
import numpy as np
import matplotlib.pyplot as plt

This example will be class based, hence the abc imports.

First, let's define our base regression class so that we have some standard methods that other regression classes can inherit:

class Regression(ABC):
    @abstractmethod
    def fit(self, X, y):
        raise NotImplementedError
    
    @abstractmethod
    def predict(self, X):
        raise NotImplementedError

Now we can implement our linear regression class:

First, let's define the constructor:

class LinearRegression(Regression):
    def __init__(self):
        self.w0 = None
        self.w1 = None

We set the weights to None initially and will update them in the fit method. As we have an analytical solution, we don't need to be careful about the initialization like with iterative methods.

Fitting the model

The fit method takes in the input features X and target values y and uses them to calculate the optimal weights. We will also specify two helper functions to make the implementation cleaner:

def fit(self, X, y):
    self.w1 = self._learn_w1(X, y)
    self.w0 = self._learn_w0(X, y)

Now we can implement the two helper functions using the two formulas derived earlier:

def _learn_w1(self, X, y):
    x_bar = np.mean(X)
    y_bar = np.mean(y)
    numerator = np.sum((X - x_bar) * (y - y_bar))
    denominator = np.sum((X - x_bar) ** 2)
    return numerator / denominator if denominator != 0 else 0

def _learn_w0(self, X, y):
    x_bar = np.mean(X)
    y_bar = np.mean(y)
    return y_bar - self.w1 * x_bar
important

The order in which we call the two helper functions is very important. We must call _learn_w1 before _learn_w0 because _learn_w0 depends on the value of self.w1.

Making predictions

After fitting, prediction is straightforward:

def predict(self, X):
    return self.w0 + self.w1 * X

And with that, the complete class should look something like this:

class LinearRegression(Regression):
    def __init__(self):
        self.w0 = None
        self.w1 = None

    def fit(self, X, y):
        self.w1 = self._learn_w1(X, y)
        self.w0 = self._learn_w0(X, y)

    def _learn_w1(self, X, y):
        x_bar = np.mean(X)
        y_bar = np.mean(y)
        numerator = np.sum((X - x_bar) * (y - y_bar))
        denominator = np.sum((X - x_bar) ** 2)
        return numerator / denominator if denominator != 0 else 0

    def _learn_w0(self, X, y):
        x_bar = np.mean(X) 
        y_bar = np.mean(y)
        return y_bar - self.w1 * x_bar

    def predict(self, X):
        return self.w0 + self.w1 * X

Applying this to our school example

If we go ahead and apply our model to our school experiment data, we find that the optimal parameters are roughly $ w_0 = 80 $ and $ w_1 = 6 $ (rounded to integers). Which means the line of best fit is given by:

$$ y = 6x + 80 $$

And the loss (e.g the error between the line and the data points) is around 20 (using the rounded values). Looking at the plot below, it looks like a pretty good fit, the line is close to all the data points.
